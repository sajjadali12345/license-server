# موقع الاشتراكات — دليل النشر من الموبايل

## شنو بالمجلد
- `index.html` — الموقع كامل (صفحة وحدة، ما يحتاج بناء أو تنصيب)
- `scripts/` — ملفات السكربتات الأربعة الجاهزة للتحميل

## خطوات النشر (بدون كمبيوتر)

### 1. ارفع المجلد لـ GitHub
- سوي repository جديد اسمه مثلاً `subscription-site`
- ارفع `index.html` ومجلد `scripts` كامل بنفس البنية

### 2. انشر بموقع استضافة ملفات ثابتة مجاني
أسهل خيار: **Netlify** أو **Vercel** أو **GitHub Pages** (الثلاثة مجانية للمواقع البسيطة زي هذا):
- Netlify: افتح netlify.com من المتصفح → سجل دخول بـ GitHub → "Add new site → Import from GitHub" → اختار الـ repo → Deploy
- بعد النشر تاخذ رابط عام مثل `https://your-site.netlify.app`

### 3. اربط سيرفر التراخيص
افتح `index.html` بمحرر الملفات (نفس اللي فتحناه بـ Replit، أو محرر نصوص عادي)، ولگه هذا السطر قريب نهاية الملف:

```js
const LICENSE_SERVER = 'https://YOUR-LICENSE-SERVER.example.com/api/verify';
```

غيّره لرابط سيرفر التراخيص الحقيقي اللي نشرناه سابقاً (من مشروع `license-server` على Railway)، مثلاً:

```js
const LICENSE_SERVER = 'https://license-server-production.up.railway.app/api/verify';
```

احفظ، وارفع النسخة المحدثة (git push، أو Netlify يحدث تلقائياً لو مربوط بـ GitHub).

### 4. جرب الموقع
- افتح الرابط اللي طلع لك
- جرب زر "اشترك الآن" — لازم يفتحلك واتساب برسالة جاهزة
- سوي ترخيص تجريبي من لوحة إدارة سيرفر التراخيص (زي ما شرحنا بالمحادثة السابقة)، وجرب تدخل المفتاح بقسم "تفعيل السكربت" — لازم يطلعلك رابط تحميل السكربت

## ملاحظة مهمة
حالياً "اشترك الآن" يفتح واتساب فقط (تأكيد دفع يدوي). لما توصلك موافقة Qi Card وتربط بوابة الدفع، نقدر نطور الزر يسوي كل شي أوتوماتيكي (دفع → مفتاح ترخيص فوري بدون ما تتدخل يدوياً).
